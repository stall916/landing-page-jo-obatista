# EcoDoa

## Descrição
EcoDoa é uma ideia de aplicação web que ajuda a reduzir o desperdício de alimentos. 
A plataforma conecta pessoas e empresas que querem doar alimentos com instituições 
ou pessoas que precisam, facilitando a doação e ajudando a combater a fome.

## ODS da ONU
Este projeto se relaciona com o **ODS 2 – Fome Zero e Agricultura Sustentável**, 
pois busca diminuir a fome e melhorar o acesso a alimentos.
